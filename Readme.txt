Steps to test project
-------------------------------------
1. Set Queue Name in UiPath Orchestrator as defined in Config.xls
2. Create CSV file with name “DonationDetails.csv” as defined format like Name, Email, Phone, Address, Country, DOB, etc.
3. Set Input File Path with FileName in Assets of UiPath Orchestrator.
4. Set Output File Path with FileName in Assets of UiPath Orchestrator.
5. Start UiPath Robot.
